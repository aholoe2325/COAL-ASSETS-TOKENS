# Coal Assets Token (CAT)

A world-class, institutional-grade Web3 platform for tokenized coal reserves. Sharia-compliant, asset-backed, and designed for ethical energy investment.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🏗️ Project Structure

```
src/
├── app/              # Next.js App Router pages
│   ├── page.tsx     # Landing page
│   ├── auth/        # Authentication pages
│   └── dashboard/   # Dashboard pages
├── components/
│   ├── landing/     # Landing page components
│   ├── dashboard/   # Dashboard components
│   └── ui/          # Reusable UI components
├── styles/          # Global styles
└── lib/             # Utilities and helpers
```

## 🎨 Design System

### Colors
- **Primary Dark**: `#0f0f0f` (cat-dark)
- **Secondary Dark**: `#1a1a1a` (cat-charcoal)
- **Accent Gold**: `#d4af37` (cat-gold)
- **Success Green**: `#10b981` (cat-emerald)
- **Text**: `#f5f5f5` (cat-text)

### Components
- **Glassmorphism**: Subtle transparent cards with backdrop blur
- **Animations**: Smooth transitions with Framer Motion
- **Typography**: Modern, clean, trustworthy aesthetic

## 📋 Features

### Landing Page
- Hero section with compelling call-to-action
- What is CAT section with feature cards

- How it works step-by-step flow
- Ethics & sustainability section
- Final CTA section

### Authentication

### Dashboard
- Overview with key metrics

- Profit Distribution history
- Transaction history
- DAO-style governance
- Settings and profile management

## 🔧 Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS with CSS variables
- **Animations**: Framer Motion
- **State**: Zustand (ready for integration)
- **Charts**: Recharts (ready for integration)
- **Icons**: Lucide React
